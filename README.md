# SmartStream: Documents Repository

[cite_start]This repository serves as the central host for all formal documentation, design artifacts, and meeting records for the SmartStream Integrated Business Management System[cite: 3, 410].

## Contents
* [cite_start]**pitch/**: Contains the project Idea Pitch deck (PDF) and the signed Project Approval Form[cite: 999].
* [cite_start]**srs/**: Software Requirements Specification (SRS) documents, including markdown sources and PDF exports[cite: 410, 999].
* [cite_start]**sds/**: Software Design Specification (SDS) documents covering architectural and detailed design[cite: 410, 999].
* [cite_start]**architecture/**: Contains Architectural Decision Records (ADRs) justifying technology and pattern choices[cite: 927, 999].
* [cite_start]**diagrams/**: Source files (.drawio, .puml) and exported images (SVG/PNG) for all UML diagrams[cite: 410, 982, 989].
* [cite_start]**meetings/**: Formal meeting notes and team log forms, committed within 24 hours of each session[cite: 834, 999].
* [cite_start]**planning/**: Project timeline and Gantt charts[cite: 999].

## Tooling
* [cite_start]**Draw.io / PlantUML**: Required for viewing or editing diagram source files[cite: 982, 983].
* [cite_start]**Markdown Editor**: Recommended for updating documentation and ADRs[cite: 927, 999].
